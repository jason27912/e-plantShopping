# 🌿 Paradise Nursery - Plant Shop E-Commerce

## Project Name
**Paradise Nursery** - An e-commerce web application for buying indoor plants.

## Project Details
This is a React-based e-commerce application that allows users to:
- Browse a collection of indoor plants grouped by categories
- Add plants to a shopping cart
- Update quantities in the cart
- Remove items from the cart
- View total items and total cost

## Technologies Used
- React 18
- Redux Toolkit for state management
- React Router DOM for navigation
- CSS for styling

## Features
- Landing page with company information
- Product listing page with 6+ plants in 3+ categories
- Shopping cart with quantity controls
- Persistent cart state using Redux
